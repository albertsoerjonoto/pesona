## 1. TL;DR

**Verdict: SHIP-WITH-FIXES.** The repo is further along than the docs suggest — RLS is clean, Midtrans webhook is properly signed/idempotent, photo-analysis has a real clinical-term scrubber with tests, onboarding is one-question-per-screen, weekly-report has a working Instagram share-card, and a referral system is wired. But **three P0s must land before public launch**, and most of the remaining gap between "functional" and "MEDvi-killer" is growth surface-area (share cards, trust signals, paywall reach) rather than plumbing.

- **P0 #1 — Coach chat has no clinical-term validator.** `/api/coach/route.ts` never calls anything like the `validateOutput()` helper that `/api/analyze-photo` uses. The system prompt blocks 6 terms; the photo route's post-filter blocks 24. A Gemini hallucination in chat will ship verbatim. (Spec §5.3 / §7.3 mandate belt-and-suspenders.) — [`src/app/api/coach/route.ts:185-263`](src/app/api/coach/route.ts:185)
- **P0 #2 — No chat rate-limiting.** `src/lib/payments/rate-limit.ts:32` defines `checkRateLimit()`; coach route never imports it. Free-tier users can send unlimited messages. Spec §10.1 mandates 3/day. Also unprotected LLM-cost ceiling. — [`src/app/api/coach/route.ts`](src/app/api/coach/route.ts) (no import)
- **P0 #3 — No Haloskin escalation CTA.** Spec §5.5 prescribes a verbatim Bahasa template + a Haloskin booking button ("Rp 25.000–50.000 per konsultasi"). Current coach prompt offers only "konsul ke dermatologist" as prose; chat UI has no escalation branch; "Haloskin" appears nowhere in `/src`. — [`src/app/api/coach/route.ts:29`](src/app/api/coach/route.ts:29), no renderer in [`src/app/(app)/chat/page.tsx`](src/app/(app)/chat/page.tsx)
- **Biggest growth miss:** share-card infrastructure exists ([`src/app/(app)/reports/page.tsx:236-284`](src/app/(app)/reports/page.tsx:236) uses `html-to-image` → `navigator.share` with Instagram fallback) but is confined to weekly reports. No share on streak milestones, progress photos, or referrals. The TikTok/IG loop is one plumbing pass from being lit.
- **A 10/10 MEDvi-killer would:** (a) ship the three P0s, (b) add share-cards to 5 more surfaces, (c) put a trust counter + one real hijabi before/after strip above the landing fold, (d) wire the Haloskin booking redirect, (e) add a phone-OTP sign-in. None require net-new plumbing — all use primitives already in the tree.
